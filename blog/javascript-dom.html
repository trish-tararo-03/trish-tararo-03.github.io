Control Flow and Loops - Using an example from everyday life

Control flow is simply the order in which a computer will execute the statements, instructions or functions written. It is usually executed from top to bottom.

A good example (That I really want to incorporate into my life) is going to the gym. The process may look something like this:

  • Pack your gym bag
  • Head to the gym
  • Workout
  • Cooldown
  • Leave the Gym

Generally, the process of going to the gym wouldn’t change. This would be considered the control flow.

Where the flow could change however is during the actual workout. We could introduce a loop to dictate how many sets we do before we leave the gym. A loop is simple set of instructions that has some conditions. In this example, the condition is how many times we do a single set. Lets say we are working on our core. One set could look like:

  • Crunches x 20
  • Hip Dips x 45sec
  • Leg Raise x 45secs

The process would then look like:

  • Pack your gym bag
  • Head to the gym
  • Workout
    • Do One Set of core exersizes
    • Question why you like pain while doing another set
    • Finish strong with One. More. Set!
  • Cooldown
  • Leave the Gym

Using the loop, we decided to do three sets before we leave the gym. The loop would run through each set untill it has been fufilled, and we can say that we are one step closer to the defined six pack we all crave for summer!
